```py
## 탐색연산 
def search(self,key
  v= self.head # dummy node
  while v.next != self.head: # v가 리스트 한바퀴를 돌떄 까지 반복한다
    if v.key == key:
       return key
    v= v.next
  return None
```
삭제할때는 내가지우고 싶은 `key`값을 바로 `remove` 함수를 사용하는게 아니라
`search`사용하여 노드의 위치값을 가져와서 `remove`함수를 사용해 노드를 삭제한디
